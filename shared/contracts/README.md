# Gedeelde contracten

Deze map wordt de bron voor interfaces en datamodellen tussen device, backend en frontend.

## Beoogde contracten

- audiobestand met metadata bij een opname;
- aanvraag en status van verwerking;
- transcriptsegmenten met spreker en tijdsaanduiding;
- samenvatting, afspraken, actiepunten en eventuele open vragen;
- foutresponsen.

## Afspraken

- Houd contracten klein en onafhankelijk van programmeertaal of framework.
- Leg formaat, verplichte velden en voorbeeldwaarden vast.
- Bespreek breaking changes met alle betrokken teams.
- Voeg voorbeeldpayloads pas toe zodra een integratiepad gekozen is.

Er zijn nog geen contracten geïmplementeerd.
